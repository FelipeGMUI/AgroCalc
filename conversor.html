🔢 Conversor de Sistemas de Numeração

Converta valores de sensores entre Decimal, Binário e Hexadecimal — aplicado ao agronegócio digital

💻 Introdução à Computação — Sistemas de Numeração: Os computadores operam internamente em binário (base 2). Sensores agrícolas transmitem dados em hexadecimal (base 16) para compactação. O decimal (base 10) é a notação humana. Dominar a conversão entre esses sistemas é fundamental para o desenvolvimento de sistemas embarcados no agro.

🗺️ Mapa de Produtividade — Talhões

Visualize e analise a produtividade por setor utilizando matrizes NumPy aplicadas ao mapeamento agrícola

📐 Matemática Aplicada — Matrizes: Cada talhão (setor da fazenda) é representado como um elemento da matriz. As operações matriciais com NumPy permitem calcular a média geral, a produtividade máxima e mínima, a média por linha (zona) e estimar a produção total da propriedade.

Configurar Matriz de Talhões

Código Python — matriz_talhoes.py

# matriz_talhoes.py
# Representa produtividade (sacas/ha) de uma fazenda em matriz
# Matemática Aplicada — Matrizes e Operações com NumPy

import numpy as np

# Matriz 3x4: 3 zonas de talhões, 4 setores cada
# Linhas = zonas geográficas (Norte, Centro, Sul)
# Colunas = setores (A, B, C, D)
talhoes = np.array([
    [62, 58, 71, 65],   # Zona Norte
    [70, 80, 75, 68],   # Zona Centro
    [55, 60, 58, 63],   # Zona Sul
])

print("=== MAPA DE PRODUÇÃO (sacas/ha) ===")
print(talhoes)
print()

# Operações matriciais
print(f"Média geral:             {talhoes.mean():.1f} sacas/ha")
print(f"Maior produtividade:     {talhoes.max()} sacas/ha")
print(f"Menor produtividade:     {talhoes.min()} sacas/ha")
print()

# Média por zona (axis=1 = por linha)
medias_zona = talhoes.mean(axis=1)
print("Média por zona:")
zonas = ['Norte', 'Centro', 'Sul']
for zona, media in zip(zonas, medias_zona):
    print(f"  Zona {zona}: {media:.1f} sacas/ha")

# Estimativa de produção total
# (assumindo 300 ha divididos igualmente entre os 12 setores)
ha_por_setor   = 300 / talhoes.size           # 25 ha por setor
total_sacas    = (talhoes * ha_por_setor).sum()
print(f"\nÁrea por setor:          {ha_por_setor:.0f} ha")
print(f"Estimativa total (300ha): {total_sacas:,.0f} sacas")

# Soma das matrizes (comparação com meta de 70 sc/ha)
meta = np.full_like(talhoes, 70)
diferenca = talhoes - meta
print("\nDesvio em relação à meta (70 sc/ha):")
print(diferenca)

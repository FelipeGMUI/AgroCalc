🧪 Blend de Fertilizante — Sistema NPK

Resolução de sistema linear 3×3 para cálculo das quantidades de ureia, superfosfato e KCl

📐 Matemática Aplicada — Sistemas Lineares e Determinantes: O problema de blend de fertilizantes resulta em um sistema de três equações lineares na forma A·x = b, onde A é a matriz de composição nutricional dos fertilizantes, x o vetor de quantidades (kg/ha) e b a meta de NPK. O determinante de A verifica se o sistema possui solução única.

O Sistema Linear NPK

Cada fertilizante possui uma composição fixa de N, P e K. Para atingir a meta nutricional, resolvemos o sistema:

0,45·u + 0·s + 0·k = N_meta
0,10·u + 0,46·s + 0·k = P_meta
0,05·u + 0·s + 0,60·k = K_meta

Em forma matricial: A · x = b

A = [0.45 0.10 0.05]
    [0.00 0.46 0.00]
    [0.00 0.00 0.60]

Verificação com Determinante

Se det(A) ≠ 0, o sistema tem solução única. Caso contrário, as combinações de fertilizantes são linearmente dependentes e a meta NPK não é atingível com esses produtos.

Conjuntos de Insumos

Classificação por tipo (teoria de conjuntos):

• Orgânicos: {esterco, torta de mamona, pó de rocha}

• Químicos: {ureia, superfosfato, KCl, MAP, DAP}

• Híbridos: {bioestimulantes, composto enriquecido}

• Orgânicos ∩ Químicos = ∅ (conjuntos disjuntos)
